## 快速开始

### 环境要求

- Node.js >= 16 / Python >= 3.8
- Docker >= 20.10（可选）

### 安装运行

```bash
# 克隆项目
git clone https://github.com/LNU-SecLM/项目名.git
cd 项目名

# 安装依赖
npm install
# 或
pip install -r requirements.txt

# 配置环境
cp .env.example .env
# 编辑 .env 填入必要配置

# 启动项目
npm run dev
# 或
python main.py
